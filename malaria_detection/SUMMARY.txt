```

### 2. Organize Data (2 minutes)
```
data/raw/
├── Parasitized/  (infected cell images)

### 3. Train Model (10-30 minutes)
```bash

```

### 4. Make Predictions (1 minute)
```python

diagnoser = MalariaDiagnoser('models/best_model.h5')
result = diagnoser.predict_single_image('cell.jpg')
print(f"Diagnosis: {result['prediction']}")
print(f"Confidence: {result['confidence']:.1%}")
```

## 🎓 Deep Learning Architecture

```
Input Image (224x224x3)
    ↓
MobileNetV2 Base (Transfer Learning)
    ↓
Global Average Pooling
    ↓
Dense(256) + ReLU + Dropout(0.5)
    ↓
Dense(128) + ReLU + Dropout(0.3)
    ↓
Dense(1) + Sigmoid
    ↓
Output: Parasitized (0) or Uninfected (1)
```

**Training Configuration:**
- Optimizer: Adam (lr=1e-4)
- Loss: Binary Crossentropy
- Batch Size: 32
- Epochs: 25 (with early stopping)
- Data Augmentation: Yes
- Transfer Learning: Yes

**Expected Accuracy:** 85-95%
